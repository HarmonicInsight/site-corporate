## 概要

Pixieは、AIがタスクの優先順位付けを支援する次世代タスク管理アプリです。
Claude Haiku APIを活用し、自然な対話形式でタスクを管理できます。

## 主な機能

### 1. AIタスク分析

タスクを入力すると、AIが自動的に以下を分析します：

- **緊急度**: 今すぐ対応が必要か
- **重要度**: ビジネスへの影響度
- **所要時間**: 完了までの推定時間
- **依存関係**: 他のタスクとの関連性

### 2. スマートスケジューリング

AIがあなたの作業パターンを学習し、最適なスケジュールを提案します。
「集中力が高い午前中に重要タスク」「午後は会議やコミュニケーション」といった
パーソナライズされた提案が可能です。

### 3. 自然言語入力

以下のような自然な言葉でタスクを追加できます：

```
「来週金曜までに企画書を完成させる」
「毎週月曜に週次レポートを作成」
「田中さんからの返信待ち」
```

### 4. チーム連携

- タスクの共有と委任
- 進捗状況のリアルタイム同期
- コメントとディスカッション

## 技術スタック

- **フロントエンド**: React Native
- **バックエンド**: Node.js
- **AI**: Claude Haiku API
- **データベース**: PostgreSQL

## 対応プラットフォーム

- iOS 15.0以上
- Android 10以上
- Web版（準備中）

## まとめ

Pixieは、AIの力で日々のタスク管理をシンプルにします。
「何をすべきか」で悩む時間を減らし、「どう実行するか」に集中できる環境を提供します。
